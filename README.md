# VRM Chatbot

This project is a 3D VRM Chatbot application built with React, Three.js, and Shadcn UI.

## Getting Started

1.  **Install dependencies:**
    ```bash
    bun install
    ```

2.  **Run the development server:**
    ```bash
    bun dev
    ```

3.  **Open your browser:**
    Navigate to `http://localhost:5173` (or the port shown in the terminal).

## Features

*   **3D Avatar:** Interactive 3D VTuber avatar (VRM) rendered using React Three Fiber.
*   **Chat Interface:** Modern chat UI built with Shadcn components.
*   **Controls:** Orbit controls to view the avatar from different angles.

## Assets

*   The VRM model is located at `public/vrm/Model1.vrm`.
